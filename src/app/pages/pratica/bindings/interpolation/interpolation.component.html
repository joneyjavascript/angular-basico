Interpolation

É possível interpolar expressões dentro do template

Interpolando uma propriedade do Componente

{{ valor }}

Interpolando o retorno de um método do Componente

{{ resultado() }}

Interpolando uma expressão direto no template

{{ "5 x 5 = " + (5 * 5) }}

Interpolar uma expressão dentro de uma tag html

Interpolação com uso de pipes

{{ nome | uppercase }}

Interpolação com uso de pipes com parametros

nascimento: {{ nascimento | date:'dd/MM/YYYY' }}

saldo na conta: {{ saldoNaConta | currency }}

Interpolação com Pipe customizado (reverse)

{{ texto | reverse }}
